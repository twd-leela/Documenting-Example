# Covid 19 Data Analysis
In this real world project, we'll walk through the entire process analyzing covid-19 data using PowerBI.
We'll start by importing the dataset, which contains information on Covid-19 cases, deaths, and recoveries from around the world.
Then, we'll use PowerBI to transform the data into a format that's easy to analyze and visualize.
Next, we'll dive in to the analysis phase.
We'll use PowerBI's visualization tools to create charts, graphs, maps that will help us gain insights into the data.
We'll explore trends over time, look at the impact of covid-19 on different regions and countries, and examine the relationships between different variables in the data.
